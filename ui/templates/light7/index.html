Light7 Mobile

是一套基于 Framework7 开发的UI库。它非常轻量、精美，Light7 是一个库而不是一个框架。它提供了 20+ 个常用的组件，却只有 82K JS 和 117K CSS 大小的两个文件。大部分时候你只需把组件的HTML代码复制到自己的HTML文件中即可。并且能兼容到 iOS 6.0+ 和 Android 4.0+，方便迅速搭建手机H5应用，非常适合开发跨平台Web App。
